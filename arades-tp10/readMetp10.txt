Pour compiler le code : make
Pour exécuter le code : ./bin/exec

Pour générer le doxygen :
doxygen -g
doxygen Doxyfile

Les entrées librairie se trouvent dans le fichier tp9.h

Le programmme permet de jouer au jeu du pendu

Il faut d'abord saisir le nom du joueur

Il faut ensuite saisir les lettres que l'on veut jouer une par une, comme indiqué.
Les lettres doivent être saisies en majuscule.
